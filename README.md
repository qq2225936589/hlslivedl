# hlslivedl
Direct Download TS files from HLS Live stream

Usage: hlslivedl [options] -i [http://...m3u8]

    -o set output filename
    
    -p set proxy http: https: socks4: socks4a: socks5: socks5h:
    
    -u set useragent
    
    -d debug mode
    
    Press [Q] to stop download
